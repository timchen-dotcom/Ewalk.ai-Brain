# Ewalk.ai Brain

這是 Ewalk.ai 的 Obsidian 工作 Vault，用來管理客戶、內容、Reels、廣告、月報、素材、會議與知識沉澱。

## 使用方式

- 新想法、待整理資訊先放進 `00_收件匣`
- 每位客戶放在 `01_客戶`，建議使用 `模板/客戶模板.md`
- IG 貼文與圖文內容放在 `02_內容/IG內容`
- Reels 腳本與短影音企劃放在 `02_內容/短影音Reels`
- 廣告素材、受眾、文案與投放紀錄放在 `03_廣告`
- 月報與成效回顧放在 `04_報表/月報`
- 策略、定位、漏斗、活動規劃放在 `05_策略`
- 品牌素材、圖片、文案資產放在 `06_素材`
- 會議紀錄放在 `07_會議`
- 自動化流程、SOP、工具串接放在 `08_自動化`
- 可重複使用的洞察與方法論放在 `09_知識庫`
- 已結案或不再使用的資料放進 `99_封存`

## 建議命名

- 客戶：`客戶名稱 - 客戶檔案`
- IG 內容：`YYYY-MM-DD - IG - 主題`
- Reels：`YYYY-MM-DD - Reels - 主題`
- 廣告：`YYYY-MM-DD - 廣告 - 活動名稱`
- 月報：`YYYY-MM - 客戶名稱 - 月報`
